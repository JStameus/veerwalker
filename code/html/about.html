What is Veerwalker?

Veerwalker is a browser-based role-playing game inspired by tabletop RPGs such as Dungeons & Dragons. It is meant to emulate a tabletop gaming experience in a structured single-player format.

About the Developer

Hi! I'm Joseph. I'm an aspiring web developer/programmer with a background in game development and 3D art. I'm currently studying web development and I'm making this game as a student project, with the hope that I can continue expanding on the idea as I learn more.

Project Overview

The first "phase" of the project is only meant to include a small portion of the finished game, and I'll focus only on making the text-based dialogue and turn-based encounters. The goal is to get these two parts polished and fun enough to stand on their own, but in the future, there are quite a few more features I'm hoping to add, depending on how much time I have, and how difficult it turns out to be:

  • A finished and complete storyline
  • Tiled world overview map with settlements, random encounters and points of interest
  • Fully functional quest system
  • A simple simulated economy with crafting and trading
  • The ability to build settlements
  • Faction system
  • Endless dungeons
  • Multiplayer?!

Like I said, quite a bit. It may take years, if I even finish it at all. We'll see!
